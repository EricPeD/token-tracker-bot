# Token Tracker Bot

[![Tests](https://github.com/EricPeD/token-tracker-bot/actions/workflows/ci.yml/badge.svg)](https://github.com/EricPeD/token-tracker-bot/actions)
[![Coverage](https://img.shields.io/badge/coverage-90%25-brightgreen)](https://codecov.io/gh/EricPeD/token-tracker-bot)

Telegram bot for real-time ERC-20 deposit notifications on Polygon (Etherscan V2 / Moralis ready).

## Quick Start
1. `pip install -e .`
2. Copy `.env.example` to `.env`, add keys.
3. `python src/bot/main.py`

## Features
- /setwallet {address}
- /addtoken {contract}
- Notifications with tx details.
